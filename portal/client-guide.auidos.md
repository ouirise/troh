## Creating a Post

### 1. Complete the Form

| Field | Instructions | Example |
|-------|-------------|---------|
| **Title** | What the post is about | "My First Day of Chemo" |
| **Date** | Publication date | Today's date |
| **Author** | Your name | "Elaine Taylor" (pre-filled) |
| **Featured Image** | Drag photo or click to select | Shows preview when uploaded |
| **Excerpt** | 1-2 sentence summary | "Sharing my experience from my first chemotherapy treatment." |
| **Content** | Full blog post | Your story, formatted |

### 2. Formatting Content

Use Markdown syntax for styling:

| Effect | Syntax | Result |
|--------|--------|--------|
| Bold | `**text**` | **bold text** |
| Heading | `## Heading` | Section title |
| Paragraph | Blank line between text | Separate paragraphs |
| Images | Paste directly | Inline images |

### 3. Preview

Review the **Live Preview** panel on the right. Verify appearance before publishing.

### 4. Publish

Click **"Save Post"** — content goes live immediately.

## Best Practices

### Photography

| Recommendation | Details |
|----------------|---------|
| Format | JPG or PNG |
| Size | Under 5MB (typical phone photos acceptable) |
| Orientation | Landscape preferred |

### Writing

- **Authenticity**: Genuine stories create connection
- **Honesty**: Share complete experience — positive and challenging
- **Detail**: Include specifics that may help others

### Editing Existing Posts

1. View "Existing Posts" panel on right
2. Click eye icon to view post
3. Contact developer with JSON data for modifications

## Support

### Issue Resolution

| Problem | Solution |
|---------|----------|
| Page not loading | Check internet connection |
| Changes not appearing | Refresh and retry |
| Required field errors | Complete all fields marked with * |
| Other errors | Contact developer with error message |

### Status Indicators

| Symbol | Meaning |
|--------|---------|
| ✅ | Success — post saved |
| ❌ | Error — contact developer |
